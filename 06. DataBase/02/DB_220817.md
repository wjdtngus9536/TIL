# 1.    WHERE

### 1. 연산자
- 논리 연산자 ( 순서에 주의 )  
```sql
WHERE HEIGHT = 175 OR HEIGHT = 183 AND WEIGHT = 80;
WHERE (HEIGHT = 175 OR HEIGHT = 183) AND WEIGHT = 80;
```
> 위는 키가 175 | (키 183 & 몸무게 80)  
> 아래는 키가 175랑 183인 사람 중 몸무게 80인 사람을 뜻함
