## 01. CSS 기본 스타일

- 크기 단위

    - px : 고정적인 단위
  
    - % : 가변적인 레이아웃에서 사용(백분율 단위)

    - em : 부모 요소에 대한 상속의 영향을 받음  
    html root 16px을 기반으로 2배 크기의 h2, ul은 그 h2에 대한 2배 64px의

    - rem : 최상위 요소의 사이즈를 기준으로 배수의 단위를 가짐
      ```html
      <!-- h2.rem+ul.rem>li.rem -->
      > : sub indentation
      + : 한 줄 추가
      <h2 class="rem">2rem</h2>
      <ul class="rem">
        <li class="rem">2rem</li>
      </ul> 
      ```
